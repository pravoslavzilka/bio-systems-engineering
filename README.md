# Bio Systems Engineering

This repository contains engineering models, simulations, and analyses related to biological systems, with a focus on cultivation systems, fluid dynamics, and process optimization.

## Motivation

The goal is to develop a deeper engineering understanding of bio-systems by combining:
- physical modeling
- numerical simulations
- system-level thinking

The work is closely connected to real-world biomanufacturing problems.

## Structure

- `cultivator/` – system-level description and design
- `duckweed-model/` – population dynamics modeling (ODE)
- `notes/` – insights, assumptions, and observations

## Approach

Each problem is treated as an engineering system:
1. Define assumptions
2. Build a simplified model
3. Run simulations
4. Extract insights
5. Iterate

## Status

Work in progress – models are being continuously refined and expanded.
